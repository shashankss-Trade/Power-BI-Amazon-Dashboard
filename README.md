# 🛒 Amazon E-commerce Sales Analytics Dashboard

An interactive Power BI dashboard built to analyze Amazon E-commerce sales performance, customer behavior, product insights, and operational metrics using Power BI, Power Query, and DAX.

---

# 📊 Dashboard Overview

| Page | Description |
|------|-------------|
| 1 | Business Overview Dashboard |
| 2 | Product & Customer Analysis |
| 3 | Operations & Shipping Insights |

---

# 🛠 Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Data Modeling
- Excel / CSV Dataset

---

# 📌 Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Quantity
- Repeat Customers
- Average Revenue Per Customer
- Average Delivery Days
- Shipping Cost Analysis

---

# 📈 Business Insights

- Technology category generated highest sales
- Standard Class had highest shipping usage
- Identified top spending customers
- Analyzed repeat customer behavior
- Tracked monthly sales trends
- Evaluated operational efficiency using delivery metrics
